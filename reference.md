## Useful keyboard shortcuts

- Assignment operator `<-`: <kbd>Alt</kbd> + <kbd>-</kbd> (PC); <kbd>&#8984;</kbd> + <kbd>-</kbd> (Mac)
- Run line of code: <kbd>Ctrl</kbd> + <kbd>Enter</kbd>
- Pipe `%>%`: <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>M</kbd>
